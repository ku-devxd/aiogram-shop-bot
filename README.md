## Telegram Shop Bot (Aiogram 3 + SQLite)

A simple and modular Telegram shop bot built with **Aiogram 3**, using **SQLite** for local storage.

✅ Users can browse products  
✅ Add items to cart  
✅ Place orders  
✅ Admin panel for managing catalog (add/delete products, price, photo)

---

###  Features

- Built with **Python + Aiogram 3**
- **SQLite** database (easy to deploy, no external dependencies)
- Clean and scalable project structure
- Supports environment variables (`.env`) for secure config
- Portfolio-ready project for freelancers

---

### Tech Stack

| Component | Technology |
|----------|------------|
| Backend  | Python (Aiogram v3) |
| Database | SQLite |
| API / Bot | Telegram Bot API |

```bash
git clone https://github.com/ku-devxd/aiogram-shop-bot.git
cd aiogram-shop-bot

python3 -m venv .venv
source .venv/bin/activate     # macOS / Linux
# .venv\Scripts\activate      # Windows
